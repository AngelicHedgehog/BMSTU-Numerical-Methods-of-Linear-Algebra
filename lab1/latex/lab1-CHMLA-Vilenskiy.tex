\section{Задание}\label{Sect::task}

Установить интерпретатор Julia на рабочее устройство.
Познакомиться с базовыми инструкциями языка.
Написать программу, визуализирующую три трехмерных графика.

\section{Результаты}\label{Sect::res}

Исходный код программы представлен в листинге~\ref{lst:code1}.

\begin{figure}[!htb]
\begin{lstlisting}[language={},caption={Визуализация трехмерных функций},label={lst:code1}]
using Plots
using CalculusWithJulia
import Contour: contours, levels, lines, coordinates
using LinearAlgebra, ForwardDiff

xs = range(-2, stop=2, length=100)
ys = range(-2, stop=2, length=100)

surface(xs, ys, ((x, y) -> cos(x)*sin(y)))
surface(xs, ys, ((x, y) -> x^2-y^2))
surface(xs, ys, ((x, y) -> 1/(1+x^2)+1/(1+y^2)))
\end{lstlisting}
\end{figure}
